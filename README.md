# cheain

Демонстрационное desktop-приложение на PyQt6 для мгновенных переводов,
hash-цепочки блоков, дерева Меркла и симуляции атак на копию цепочки.

## Установка

```bash
python -m pip install -r requirements.txt
```

## Запуск

```bash
python app.py
```

## Что внутри

- Темная dashboard-компоновка с боковой навигацией.
- Создание переводов вручную и через авто-генератор.
- Автоматическое и ручное закрытие блоков.
- Визуализация цепочки и отдельное окно дерева Меркла.
- Симуляция атак на копию цепочки без изменения оригинала.
